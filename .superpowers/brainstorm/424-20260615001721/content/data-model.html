Booking System - Data Model

核心資料表關聯圖

Database Schema
User id, name, phone, email, password_hash, role(USER|VIP|SALES|ADMIN) total_spent, created_at Plan (消費計劃) id, name, play_hours, available_days(WEEKDAY|WEEKDAY_NO_HOLIDAY|ANY) available_time_start, available_time_end earliest_entry, latest_leave, latest_leave_day(SAME|NEXT) price, deposit, overtime_rate_per_hour is_active, created_at Booking id, user_id, plan_id, date, start_time, end_time persons, status(PENDING|CONFIRMED|CANCELLED) amount, deposit, total created_at Payment id, booking_id, method_id, receipt_image status(PENDING|VERIFIED|REJECTED) uploaded_at PaymentMethod (支付設定) id, name, qr_code_image, is_active Holiday (公眾假期, 來自HK Gov API) id, date, name Setting (系統設定) key, value (e.g. max_seats=10) OAuthAccount id, user_id, provider, provider_account_id

Routes / Pages

前台 (Public)
/ → Home (landing)
/auth/login → 登入
/auth/register → 註冊
/booking → 預約日曆
/booking/payment → 付款
/account → 我的帳號
後台 (Admin/Sales)
/admin → Booking 總覽
/admin/plans → 消費計劃設定
/admin/payments → 支付設定
/admin/users → 會員管理
/admin/settings → 系統設定
